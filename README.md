## Goal
Scrape real estate property data from 
titleflex.com, zillow.com, and realtor.com

## Requirements
* Python 3.7+
* Selenium FireFox Webdriver
* TitleFlex Account

## Instructions
To get address data, run  
```
python main.py <ADDRESS>
```

Example  
```
python main.py 6560 Arundel Pl, San Diego, CA 92117
```
